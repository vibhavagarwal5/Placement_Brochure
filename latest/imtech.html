Integrated MTech

The iM.Tech course curriculum consists of Core Courses and Electives spanning ten semesters.
While the same set of core courses have to be completed by all the students every semester, the students are free to choose their electives. While core courses help in creating a strong foundation in the Information Technology field, electives allow students to learn and specialize in their own areas of interest.

The core courses, categorized into Computer Science, Electronics and Communication, Basic Sciences and Miscellaneous entirely make up the first five semesters. Electives are also categorized into Computer Science, Electronics & Communication, Data Science, Networks & Communication along with other core courses, are taken up from sixth semester till the ninth semster

Though the electives of students may vary as per their interests, the number of electives per semester remains the same. The current(ninth) semester requires students to take any two electives and a twelve credit Project under the supervision of a Professor in the college. During the tenth semester, a student can either take up an industry internship or academic research at the institute leading to a thesis.

Computer Science

  • Virtual Stock Guru
    This project tries to simulate the real stock market by providing users virtual cash and letting them trade on live stock values. It provides a good platform for beginner level investors to hone their stock market skills. It is written using Ruby on Rails and Python.
  • Web for all
    This project tries to create a Heroku like setup to enable every student in college to have their own websites at .iiitb.org. This tries to cut down the requirement of Virtual Machines as well as Hardware Infrastructure and give the full control to end users who write their own code and build scripts which can then be easily deployed.
  • Animation in OpenGL using SceneGraph
    Designed and rendered a complex scene in C++ using the OpenGL library, utilizing scene-graph and MVC concepts. The scene had multiple independent and interdependent objects, various types of stationary and mobile light sources, optional textures and a GUI to let the user control various parts of the animation.
  • Security in Cyber-Physical Systems
    Used FMEA and game theory for modelling and analysing the security of cyber-physical systems with the automobile industry as a case-study. Formulated an asymmetric non-zero sum sequential game between the attacker and the defender.
  • Protoype for Facebook
    The main aim of the project is to develop a prototype of Facebook using concepts of Object Oriented Programming. The technologies involved are Java and JDBC for backend, MySQL for Data storage and management. Also, HTML and Twitter Bootstrap are used for Front-end whereas JavaScript and Jersey framework are used for REST services and servlets for integrating Front-end and Back-end.

Data Science

  • Distracted Driver Detection
    Created a neural network to detect some basic distractions that a driver can face while driving and warn him appropriately to focus on the drive. The warning was customised to be given in the voice of a close person to create an emotional connect instead of viewing it as an annoyance.
  • DNA – phred recalibration
    DNA analysis assigns a phred score to every location in the DNA. A model was created to find non-real mutations (arising from the chemical process of extraction of the DNA as compared to genuine biological mutations) from the variant callset consisting of all the mutations detected, using machine learning techniques.
  • Deception Detection - Automatic Recognition of Fake News using Adversarial IR
    Given a news article the goal of the project was to build a model which could classify the article as genuine or fake. The main algorithm used was the Weighted Majority Algorithm (WMA). WMA algorithm has experts and the experts used were neural networks and certain semantic based classifiers.
  • Cognitive Human Robot Intelligence
    Formulate a robot model that can adjust its behavior to interact and collaborate effectively with a human whose own behavior and performance are subject to unpredictable changes like mood swings, changes of emotional states, boredom etc.
  • Recognise Nametag and Rank from Army Uniform
    Detect and recognise name-tag and rank of an army soldier from a video using Image Processing and Deep Learning.
  • Machine Translation between Indian Languages Telugu-Hindi
    Machine Translation (MT) is translation of a natural language content from one language to the other. Created Statistical Machine Translation to translate between Hindi and Telugu languages. It is an automated system with greater translation accuracy ideally without any kind of human intervention.

A model was built which used the results from the traditional retrieval using Page Rank and reranked the pages based on Topic-Sensitive Page-Rank(TSPR) values given a query and a user.

Electronic System Design

  • Implementation of census transform on FPGA

This project achieved an upgrade in technology by using four LEDs for the calculation of multiple parameters like haemoglobin, spo2,heart rate, BP, etc. with better accuracy.

    Designed and built a working model of a moon rover, with software to wirelessly control it and capture pictures and video. The team built a WiFi controlled vehicle using a Raspberry Pi board and a microcontroller, and assembled simple circuits for motors. Transfer of images and control involved Python programming.

Web Sciences Lab

The Web Science Lab (WSL) at IIIT Bangalore focuses on building models to extract semantics and understand the impact of the web on different facets of human life.
Typical research pursuits of WSL include the following:

  • Mining semantic associations from text and social media data
  • Building computational models for various socio-cognitive phenomena on the web (like formation of collective opinions, creation of celebrities, etc.)
  • Semantic integration of formal web data

Computational Sciences Lab

The Computational Sciences Lab at IIIT-B is interested broadly in the areas of Algorithms, Optimization, and Robotics.
Major focus areas include:

  • Robust optimization under uncertainty, with applications to supply chains, real time search, banking, smart grid, transportation, gaming and allied areas
  • Approximation algorims, machine learning, cryptography and linguistics, Visualization (scientific and information), high performance computing, computational geometry and topology
  • Electronic Design Automation including Statistical timing analysis and Optimization for Digital circuits, Power Analysis and Optimization, Formal Verification, Semiconductor manufacturing, Statistical Optimization, Combinatorial Optimization, Design and Analysis of Alogrithms

HiDes Lab

The High Density Electronic Systems Lab at IIIT-B focuses on research and development in the area of high density and low dimensional electronics. The areas of research include 3D electronics, magnetic logic devices, interconnects, and antennas.

CEEMS Lab

The Center for Electronics and Embedded Systems (CEEMS) Lab's objective is to nurture talent by focusing on Embedded Computing, Wireless Communication and Computer Vision. CEEMS Lab collaborates with public and private organizations with the aim of bridging the gap between academic output and industry requirements, thus providing every learner an equal opportunity to become industry ready.

Wireless Network Lab

This lab focuses on research involving networking standards and technology. Current research work concentrates upon the latest broadband wireless access technology – WiMAX/LTE. Major areas of work include medium access control (MAC), QoS, QoE, power management, media independent handover (MIH).
